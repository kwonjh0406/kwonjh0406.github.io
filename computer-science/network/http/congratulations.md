# HTTP 상태 코드

## HTTP 상태 코드란 무엇인가?

클라이언트의 HTTP 요청에 대한 서버의 응답 상태를 알려주기 위한 코드입니다.

## HTTP 상태 코드의 종류

맨 앞의 숫자에 따라 응답의 종류가 나뉩니다.
이후 두 자리는 응답에 대한 세부적인 내용을 다룹니다.

### 1xx 

정보입니다.

---

### 2xx

클라이언트의 요청이 성공적으로 처리되었음을 의미합니다.

---

### 3xx

리다이렉션입니다.

---

### 4xx

클라이언트 에러입니다.

#### 대표적인 응답 코드는 다음과 같습니다.

- 404 - Not Found: 요청한 자원(페이지)를 찾을 수 없는 상황입니다.

---

### 5xx

서버에 에러가 발생하였음을 의미합니다.
